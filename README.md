# Titanic Data Analysis

## Project Overview
The project demonstrates core data manipulation skills using Python's `pandas` library. It is divided into two main parts:
1. **Foundation:** Creating a custom DataFrame from a dictionary with custom indexing.
2. **Real-World Analysis:** Exploring, cleaning, analyzing, and filtering the famous Titanic dataset.

## Tech Stack
* **Language:** Python
* **Library:** Pandas
* **Environment:** Jupyter Notebook / Kaggle

## Key Insights (Titanic Dataset)
Based on the data analysis performed using `.groupby()`, here are the answers to the core project questions:

* ** Who was more likely to survive?**
Females were drastically more likely to survive. Their survival rate was roughly **74%**, compared to males who had a survival rate of roughly **18%**.

* ** Did class affect survival?**
**Yes, immensely.** Passengers in 1st Class had the highest survival rate (~62%), while 3rd Class passengers had the lowest survival rate (~24%). Wealth and status played a huge role in access to lifeboats.

* ** Were children prioritized?**
**Yes.** The "Child" age group (0-12 years old) had the highest survival rate of any age group (~50.6%), showing that the rule of "women and children first" was heavily enforced.

* ** What combination had the highest survival rate?**
**1st Class Females.** Being female gave you the highest chance of survival, and being in 1st class gave you the best access to lifeboats. Combining the two resulted in a near 96% survival rate!
